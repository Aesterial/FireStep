# Ассеты И Источники

## Текущее состояние проекта

На момент этой версии проект остаётся самодостаточным и не использует сторонние игровые ассеты.

## Что реально используется

- Встроенные примитивы Godot:
  - `BoxMesh`
  - `CylinderMesh`
  - `SphereMesh`
- Встроенные 3D-узлы и физика:
  - `StaticBody3D`
  - `Area3D`
  - `CollisionShape3D`
  - `Label3D`
  - `OmniLight3D`
  - `DirectionalLight3D`
- Встроенные UI-элементы:
  - `Control`
  - `PanelContainer`
  - `Label`
  - `Button`
  - `ColorRect`
  - `CanvasLayer`

## Файлы из репозитория

- `icon.svg` — иконка проекта, уже присутствовавшая в репозитории
- `case.pdf` — описание кейса, использованное как источник требований

## Подготовка под будущие ассеты

В проект добавлены:

- `assets/models/`
- `assets/textures/`
- `assets/audio/`
- `docs/ASSET_PIPELINE.md`

Также в сценах предусмотрены export-поля для подстановки визуальных сцен с моделями без изменения основной логики.

## Примечание

Если позже ты добавишь внешние модели, текстуры или звук, их источник и лицензия должны быть дописаны в этот файл отдельным списком.
