# Импорт И Замена Ассетов

## Цель

Проект уже играбелен на встроенных примитивах, но структура подготовлена так, чтобы ты мог заменить базовую геометрию на свои модели без переписывания логики сценария.

## Куда класть файлы

- `res://assets/models/` — модели `.glb`, `.gltf`, `.obj`, собранные `.tscn`
- `res://assets/textures/` — текстуры и материалы
- `res://assets/audio/` — звуки тревоги, шагов, интерфейса

## Рекомендуемый практический способ

1. Скопируй модель в `assets/models/`.
2. Дождись импорта в Godot.
3. Открой модель и, если нужно, сохрани её как отдельную сцену `.tscn`.
4. Выбери нужную игровую сцену.
5. На корневом узле в Inspector присвой импортированную сцену в export-поле.

## Какие export-поля уже подготовлены

### Цех

Сцена: `res://scenes/Main.tscn`

- `workshop_props_scene`
  Используй для декоративных пропсов цеха: паллеты, ящики, ограждения, трубы, шкафы.
- `machine_visual_scene`
  Используй для визуальной замены компрессора. Логика дыма, аварийной кнопки и интерактивных зон останется сверху.

### Эвакуация

Сцена: `res://scenes/Evacuation.tscn`

- `corridor_visual_scene`
  Используй для деталей эвакуационного коридора или двора.
- `assembly_visual_scene`
  Используй для оформления пункта сбора: стойка, табличка, ограждение.

## Что лучше делать

- Делать одну чистую `.tscn` на каждый крупный набор ассетов.
- Оставлять pivot модели в разумной точке, чтобы сцена вставала без долгой ручной коррекции.
- Держать текстуры рядом с моделью или в `assets/textures/`.
- Проверять масштаб сразу после импорта.
- Для MVP использовать low-poly или mid-poly модели без тяжёлых материалов.

## Что не стоит делать

- Не заменяй интерактивные панели и зоны на модели без коллизий.
- Не встраивай в импортированные ассеты свою игровую логику, если она уже есть в проекте.
- Не подключай сторонние плагины ради импорта обычных `.glb`.

## Минимальный быстрый апгрейд визуала

Если хочешь быстро улучшить внешний вид без полной переработки:

1. Подмени компрессор через `machine_visual_scene`.
2. Добавь набор пропсов через `workshop_props_scene`.
3. Положи 1-2 тревожных звука в `assets/audio/`.
4. Добавь декор коридора и пункта сбора в сцене эвакуации.

Этого уже достаточно, чтобы проект выглядел заметно лучше при сохранении всей текущей логики.
